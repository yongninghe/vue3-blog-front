# blog-front
一个后端程序员用于学习前端技术的工程。包括vue3 、elementplus、axios、mockjs等。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
